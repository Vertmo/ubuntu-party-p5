Lors de cette activité, tu vas pouvoir apporter des modifications au jeu Flappy Smile que j'ai réalisé, et qui n'est pour le moment pas le jeu de l'année. Je compte sur toi pour améliorer le jeu et le rendre plus amusant !
Tu peux accéder au jeu à cette adresse : https://editor.p5js.org/Vertmo/sketches/rydjVgRnm, mais tu devrais normalement déjà avoir un onglet ouvert. Commence par tester le jeu (en cliquant sur le bouton play en haut à gauche, puis en cliquant dans la fenètre de jeu. Tu peux faire monter ton smiley en appuyant sur la flêche du haut de ton clavier: essaie d'éviter les murs !
Maintenant que tu as une idée du jeu (et de ses défauts) tu peux choisir une des améliorations à réaliser dans la liste ci-dessous; ne t'en fais pas, tu sera guidé pour chaque amélioration. Je te conseille de commencer par les améliorations de la catégorie Pour commencer, puis si tu as le temps, essaie une de la catégorie Plus difficile.

Pour commencer

Couleurs et dessin
Dans cette partie, ton but sera de modifier les couleurs et le visuel du jeu. Pour se faire, tu pourra utiliser les deux fonctions ci dessous pour manipuler les couleurs. Essaie de trouver ces fonctions dans le code et de les modifier pour changer les couleurs affichées. 

background(r, g, b); // Change la couleur du fond
stroke(r, g, b); // Change la couleur de dessin
fill(r, g, b); // Change la couleur de coloriage
Tu peux utilise le sélecteur ci-dessous pour choisir une couleur:
Pour aller plus loin: en informatique, les couleurs sont codées au format RGB (ou Rouge Vert Bleu) : ce format est constitué de trois chiffres entre 0 et 255 indiquant les quantitées de rouge, vert et bleu dans la couleur. Par exemple 255 0 0 est rouge pur, 0 255 255 est cyan et 255 0 255 est violet. Il est important de noter que 0 0 0 est noir et 255 255 255 blanc !
Tu pourrais également modifier l'apparence du smiley que tu joues. Pour cela je te conseille de modifier la partie: 

function dessinerSmiley(x, y) {
  fill(255,255,0);
  ellipse(x, y, 20, 20);
  noFill();
  line(x, y-3, x, y);
  line(x+4, y-3, x+4, y);
  arc(x+2, y+2, 10, 10, 0, PI);
}
Tu pourrais par exemple remplacer ellipse(x, y, 20, 20) par rect(x-10, y-10, 20, 20) pour lui faire la tête au carré ! Ou bien remplacer arc(x+2, y+2, 10, 10, 0, PI) par arc(x+2, y+7, 10, 10, PI, 0) pour le rendre triste :(
Difficulté du jeu (vitesse des murs et taille du trou)
Tu as plusieurs solutions pour modifier la difficulté du jeu: tu pourrais par exemple essayer de diminuer la vitesse à laquelle les murs arrivent vers toi. Je te conseille dans ce cas de modifier la partie: 

function majMurs() {
  [...]
  murs[i][0] -= 8;
  [...]
}
Une autre solution que tu pourrais utiliser pour rendre le jeu plus ou moins facile, serait de modifier la taille du trou central par lequel ton personnage doit passer. Pour cela essaie de modifier la partie: 

function genMur() {
  [...]
  append(murs, [width+30, r, 80]); // x, centre, hauteur (du trou)
}

Plus difficile

Ajout et affichage d'un score
Si tu veux ajouter un score, il va falloir faire deux choses: trouver un moyen de comptabiliser ce score, et d'afficher ce score. Commence par ajouter une variable "score" au début du programme: 

var score = 0;
Comme tu peux le voir, le score est initialisé à 0: nous allons maintenant ajouter une instruction pour augmenter le score au cours de la partie. Tu devrais ajouter cette instruction quelque part dans la fonction draw(), pour qu'elle soit exécutée régulièrement. 

score++;
Maintenant, nous voulons afficher ton score ! A la fin de la fonction draw(), ajoute ces trois instructions : 

fill(0,0,0);
textSize(32);
text(score, 500, 30);
Si tu lances le jeu maintenant, tu verra qu'il y a bien un score affiché dans le coin en haut a gauche ! Cela dit, il manque toujours un petit détail : quand le joueur perd, nous voudrions que le score soit réinitialisé. Pour cela tu peux ajouter l'instruction ci-dessous à la fonction perdre(): elle remettra ton score à 0. 

score = 0;
