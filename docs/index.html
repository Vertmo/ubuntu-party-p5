Structure

function setup() {
    // Des choses qui s'exécutent au début
}

function draw() {
    // Des choses qui se répètent
}
La majorité de ton code doit être écrite dans ces deux fonctions. Le code dans setup s'exécutera une seule fois au début du programme, donc utilises le pour faire les réglages au début ! Le code de draw lui s'exécute en continu, c'est donc la que tu mettra les instructions de dessin !
Créer un sketch

createCanvas(640, 480);
Cette simple instruction permet d'ouvrir une fenêtre graphique, ou "canvas". Les deux paramètres 640 et 480 permettent de spécifier la largeur et la hauteur de la fenêtre en pixels. Cette instruction ne doit être exécutée qu'une seule fois, c'est pour cela qu'elle est placée dans setup.
Pour aller plus loin: le pixel est l'unité traditionnelle de taille des images en informatique. L'écran que tu regardes en ce moment possède une résolution maximum (en pixels), donc si tu regardes assez bien tu verras que l'image sur l'écran est en fait constituée de petits carrés de différentes couleurs: ce sont les pixels !
Couleurs

stroke(0, 0, 0); // Choisir la couleur de dessin
fill(255, 255, 255); // Choisir la couleur de remplissage
Les deux instructions ci-dessus permettent de manipuler les couleurs : stroke permet de changer la couleur avec laquelle les formes seront dessinées (ici en noir) et fill permet de changer la couleur avec laquelle les formes seront remplies ou "coloriées" (ici en blanc).
Pour aller plus loin: en informatique, les couleurs sont codées au format RGB (ou Rouge Vert Bleu) : ce format est constitué de trois chiffres entre 0 et 255 indiquant les quantitées de rouge, vert et bleu dans la couleur. Par exemple 255 0 0 est rouge pur, 0 255 255 est cyan et 255 0 255 est violet. Il est important de noter que 0 0 0 est noir et 255 255 255 blanc !
Dessiner des formes

ellipse(50, 50, 80, 80); // Dessiner une ellipse en haut à gauche
ellipse(mouseX, mouseY, 80, 80); // Dessiner une ellipse sur le curseur de la souris
L'instruction ellipse(x, y, r1, r2) permet comme son nom l'indique de dessiner une ellipse. En particulier, un cercle est une ellipse avec r1 et r2 égaux. Les deux premiers paramètres (x et y) contrôlent où l'ellipse sera dessinée sur l'écran, et r1 et r2 contrôlent la taille de l'ellipse. Fait varier les paramètres de la première instruction pour voir les changement !
La deuxième version de l'instruction (que tu devra utiliser pour compléter ton projet) dessine l'ellipse là ou pointe ta souris.
Pour aller plus loin: bien sur, il y a beaucoup d'autres formes que tu peux dessiner ! Experimentes avec rect(120, 70, 40, 80); ou triangle(30, 75, 58, 20, 86, 75); et essaie de comprendre comment ces instructions fonctionnent !
Conditions

if(mouseIsPressed) {
  // Ce code sera exécuté si le bouton de la souris est pressé
} else {
  // Celui la sera exécuté si il n'est pas pressé
}
La structure if...else te permet d'accomplir une action ou une autre en fonction d'une condition. Ici le code dans if() { ... } s'exécutera quand tu appuieras sur le bouton de ta souris, et le code dans else { ... } quand tu n'appuieras pas. Bien sur pour voir la différence, il faudra que le if...else s'exécute plusieurs fois, donc essaie de le placer dans draw.
Tu sais tout ce dont tu as besoin pour réaliser ton projet ! Essaie de combiner les éléments ci-dessus pour reproduire l'exemple montré au début de la séance.
Si tu es vraiment trop en difficulté, un dernier mot de passe te donnera accès à la solution ci-dessous, mais essaie de la chercher par toi même d'abord !
Solution complète

function setup() {
  createCanvas(640, 480);
}

function draw() {
  if(mouseIsPressed) {
    fill(0, 0, 0);
  } else {
    fill(255, 255, 255);
  }
  ellipse(mouseX, mouseY, 80, 80);
}
