Structure

function setup() {
    // Des choses qui s'exécutent au début
}

function draw() {
    // Des choses qui se répètent
}
La majorité de ton code doit être écrite dans ces deux fonctions. Le code dans setup s'exécutera une seule fois au début du programme, donc utilises le pour faire les réglages au début ! Le code de draw lui s'exécute en continu, c'est donc la que tu mettra les instructions de dessin !
Créer un sketch

createCanvas(640, 480);
Cette simple instruction permet d'ouvrir une fenêtre graphique, ou "canvas". Les deux paramètres 640 et 480 permettent de spécifier la largeur et la hauteur de la fenêtre en pixels.
Pour aller plus loin: le pixel est l'unité traditionnelle de taille des images en informatique. L'écran que tu regardes en ce moment possède une résolution maximum (en pixels), donc si tu regardes assez bien tu verras que l'image sur l'écran est en fait constituée de petits carrés de différentes couleurs: ce sont les pixels !
Couleurs

stroke(0, 0, 0); // Choisir la couleur de dessin
fill(255, 255, 255); // Choisir la couleur de remplissage
Les deux instructions ci-dessus permettent de manipuler les couleurs : stroke permet de changer la couleur avec laquelle les formes seront dessinées (ici en noir) et fill permet de changer la couleur avec laquelle les formes seront remplies ou "coloriées" (ici en blanc).
Pour aller plus loin: en informatique, les couleurs sont codées au format RGB (ou Rouge Vert Bleu) : ce format est constitué de trois chiffres entre 0 et 255 indiquant les quantitées de rouge, vert et bleu dans la couleur. Par exemple 255 0 0 est rouge pur, 0 255 255 est cyan et 255 0 255 est violet. Il est important de noter que 0 0 0 est noir et 255 255 255 blanc !
