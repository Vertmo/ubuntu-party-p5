Pour accèder aux indices ci-dessous, vous aurez besoin des mots de passe qui se trouvent dans la boite sur la table centrale. Il est d'accéder aux indices au fur et à mesure de votre progression, et donc de venir chercher un mot de passe au moment où vous en avez besoin.
Syntaxe (exemple)

let x = 1;
x = x + 2; // Une addition
if(x > 2) {
  console.log("Salut");
  x = null;
}
